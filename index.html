This Site was built for startup, AccesSOS as a part of my Senior Team Project.
I Built a Map that visualizes Text to 911 capability throughout the United States. My Team gathered FCC data and loaded it into a MongoDB Collection. Our backend implemented an API that is used by the front end to showcase which counties have text to 911 capability.
The Map is built using Google Maps API which allowed us to implement a search feature for users to find addresses or counties on the map. The site is deployed and hosted on text911.info. The styling of the website was done by me using wireframes made by the designer.

Built with React, CSS, HTML and Google Maps API

A Day In Golden Gate Park is an interactive web experience that celebrates the famous park and three distinct park attractions—built using p5.play, html and css. The first game allows users to explore the conservatory and learn about various flowers. The second game gives the user an opportunity to test out their archery skills. Finally, the third game will take the user through Stow Lake where they can ride a bike along the water. The visuals were inspired by designer Michael Schwab's original national park's identity system.

Built using HTML, CSS, and p5.js
